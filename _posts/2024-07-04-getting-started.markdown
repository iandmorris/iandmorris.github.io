The Renesas RA8M1 is one of the first Microcontrollers to contain the Arm
Cortex-M85 core. With support for this device just having been added to
the Zephyr RTOS it seems like the ideal platform for some interesting 
projects.

![RA8M1 Block Diagram](https://www.renesas.com/sites/default/files/ra8m1-block-diagram.png)

But where to start...

Luckily the Zephyr Project provides some excellent getting started material on
their website. By following the [Getting Started Guide](https://docs.zephyrproject.org/latest/develop/getting_started/index.html) I was able to get
everything installed (the Zephyr RTOS and all the tools needed to build applications) on my MacBook. However, I was left in the position of having to
do a lot of things via the command line. I've gotten more comfortable with this over the years but would still prefer to use an IDE to edit, build and debug my code. 

After searching online I found it was very straightforward to get VS Code to do pretty much everything I wanted! Following are the steps I took to get to VS Code configured so that I can build and debug Zephyr based applications on a RA8 Microcontroller.
